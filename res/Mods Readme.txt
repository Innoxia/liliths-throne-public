Please Note:
The folder structure for mods is as follows:
Clothing: \res\mods\your name\items\clothing\the item name\files Sample:rental_mommy in Innoxia's clothing mods
Weapons: \res\mods\your name\items\weapons\the item name\files Sample:dagger in weapons\Innoxia\dagger (Added to core game)
Items: \res\mods\your name\items\items\the item name\files
Sex Actions: \res\mods\your name\sex\the action\files
