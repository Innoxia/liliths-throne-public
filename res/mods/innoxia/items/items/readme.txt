For a commented example of how to make modded items, look in: 'res/items/innoxia/pills/fertility.xml'

The items in the 'race' folder can serve as examples on how to make race-related items.

When making modded items, you folder structure should mimic this 'example' directory, so it should look something like:
'res/mods/your_name/items/items/item_type_name/modded_item.xml'
