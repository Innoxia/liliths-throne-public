NOTE FOR THIS VERSION: If you're loading in from a pre-v0.4.2.5 save into this version, then the loading time might take a little longer than usual as the old save format needs to be converted into a new offspring save format. After a long load, it might be a good idea to make a new save right away, to avoid this delay next time the game is loaded.
